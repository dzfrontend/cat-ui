# matcha-ui

文档地址

matcha-ui是一个vue ui组件库，现在只有button、toast简单组件，但有一整套组件库的基础架构，后续争取做更多手机端的vue ui组件库。

# 安装

npm i --save matcha-ui

# Quick Start


import { MatButton } from 'matcha-ui'
import 'matcha-ui/dist/index.css'
export default {
  components: {
    MatButton
  }
}

# 特点

使用 Travis CI 进行持续集成 有丰富的单元测试
