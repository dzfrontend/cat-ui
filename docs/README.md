# matcha-ui

[![Build Status](https://travis-ci.org/staticdeng/matcha-ui.svg?branch=master)](https://travis-ci.org/staticdeng/matcha-ui)

## Install

```bash
npm i --save matcha-ui
```

## Quick Start

```js
import { Button } from 'matcha-ui'
import 'matcha-ui/dist/index.css'
export default {
  components: {
    Button
  }
}
```
